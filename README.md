# OrthoAutoAI

**OrthoAutoAI** is a powerful, lightweight, AI-driven spelling and grammar corrector for Windows. Built with AutoHotkey and powered by Google's Gemini 2.5 Flash model, it works universally across all applications—Notepad, Word, Discord, Browsers, and more.

## Link to the website
[orthoAutoAi.com](https://daith-42.github.io/orthoAutoAi/)

## ✨ Features

- **Universal Correction**: Select text anywhere, press a hotkey (Default: `Alt + C`), and watch it get fixed instantly.
- **AI-Powered**: Uses Google Gemini 2.5 Flash for context-aware corrections and high accuracy.
- **Multi-Language Support**: Fully localized in **7 languages**: English, French, Spanish, German, Italian, Portuguese, and Romanian.
- **History Tracking**: Keeps a log of your last 5 corrections in a dedicated tab.
- **Manual Confirmation**: Optional mode to review changes before applying them (`IsConfirmEnabled`).
- **Customizable**:
    -   Change Hotkeys (Correction & Settings).
    -   Toggle confirmation beep.
    -   System Tray integration.
    -   Startup and Desktop shortcuts.
- **Modern UI**: Clean configuration interface styled with Segoe UI.

## 🚀 Installation

1.  **Download OrthoAutoAI**:
    Clone this repository or download the ZIP.
2.  **Get a Gemini API Key**:
    -   Go to [Google AI Studio](https://aistudio.google.com/app/apikey).
    -   Create a free API Key.
3.  **Run the Application**:
    Double-click `OrthoAutoAI.exe` to start.
4.  **Configure**:
    -   On first run, paste your API Key.
    -   Select your language.

## 🛠️ Usage

### Quick Correction
1.  **Select** the text you want to correct.
2.  Press **`Alt + C`** (or your configured hotkey).
3.  Wait a moment for the AI to analyze and replace the text.

### Configuration
1.  Press **`Alt + W`** (or right-click the system tray icon and select "Configuration").
2.  Adjust settings:
    -   **API Key**: Update your Gemini key.
    -   **Hotkeys**: Customize shortcuts.
    -   **Language**: Switch interface language.
    -   **Options**: Toggle sound, manual confirmation, startup behavior.
3.  **History Tab**: View recent corrections.
4.  **Credits Tab**: View project info and website link.
